微蚀刻液电解系统

工艺简介

本工艺采用直接电解自动添加子液回用。

工艺特点

1、减少物料消耗。

2、减少金属排放，减轻污水处理压力。

3、设备一体化，自动化、智能化、美观、实用、易操作。

技术简介

创新性采用了高效循环工艺，与生产连接一体蚀刻液在设备循环大大降低了物料的消耗，生产出高纯铜。

工艺特点

1、再生子液质量高;各项参数完全满足工艺要求。

2、铜同回收效率高,铜纯度达到99%。

3、设备一体化,自动化、智能化、美观、实用、易操作。

技术简介

PBC线路板蚀刻工艺经开发研究形成再生循环系统，使蚀刻液全部回用，生产优质电解铜板使原产生危险废弃物的工艺成为清洁生产工艺，经济环保效益双提升。

工艺特点

1、采用旋流式三级萃取设备整齐美观。

2、蚀刻液原来组份不变，消耗不自动添加实现零排放。

3、采用全自动控制系统，操作人性化，工作更轻松。

4、成分采用物理分离，最大发挥资源的作用。

5、产生的副产品电解铜，效率高，质量好。

金回收

——— 技术简介

金回收系统是针对PCB企业和电镀金或沉镍金生产线排放的含金漂洗水进行金回收， 使用专用的吸金树脂，采用离子交换法选择性的将金从漂洗水中彻底吸附回收出来，回收金的经济效益非常可观。

电解铜产品

——— 工艺简介

用电解方法使铜在阴极沉积而得到的电解精炼铜。将粗铜（含铜99%）预先制成厚板作为阳极，纯铜制成薄片作阴极，以硫酸（H2SO4）和 硫酸铜（CUSO4）的混合液作为电解液。通电后，铜从阳极溶解成阳离子（CU）向阴极移动，到达阴极后获得电子在阴极析出纯铜（亦称电解铜）。 电解铜其实就是电解板。

概述

HBJ系列高效微生物是深圳恒邦洁环境技术股份有限公司与国内知名大学联合开发研究、培育出的具有创新性的一种应用于环境治理工程方面的菌种 HBJ系列高效微生物具有食物链完整、泥龄长、氨氮去除率高的特点，广泛适用于各活性污泥法工艺。

主要特点

a、降解氨氮的能力强，去除率可高达99?%以上，总氮的去除率可达85%，做到了氨氮与总氮均降解去除的效果。

b、该系统中微生物的食物链完整，降解有机物彻底，处理效率高，CODcr/BOD5的降解负荷可达到1-5kg/m³ · d。

c、污泥的泥龄长，因此剩余污泥产量少，是常规活性污泥的一半或更少，可降低所需的污泥处置费用。

d、可根据不同的水质选用不同的工艺，工艺选择灵活，设备更简单，硬件设施投资更少，运行费用低，故障率低。

HBJ系列高效微生物应用领域广泛，目前在多种行业均有成功应用的实例，其运用条件灵活，可以结合多种工艺进行应用

根据企业性质分类：制药、焦化、化工、印染、皮革、屠宰、水产养殖等行业。

根据条件工艺分类：好氧，厌氧，兼氧。

根据构筑物特点分类：曝气生物滤池、完全混合曝气池、流化床、SBR池、接触氧化池、UASB、IC、ABR厌氧池、水解酸化池等。

概述

实用新型多级氧化反应技术包含四相多级氧化反应器和新型强氧化反应技术这两种技术，是恒邦洁公司根据国内当前的环保形势，为了有效地解决各处理工艺降解 CODcr不彻底，去除率有限的问题而自主研发的一种针对CODcr降解的深度处理设备。该反应器内设紫外线发生装置，在处理废水时集固、液、气、光四相多位一体， 使废水的处理效率提高。

技术优势

第一、与其他深度处理技术对比：

a、降解CODcr、色度较为彻底，运行稳定性良好。

b、常温常压下反应，条件温和。

c、同等去除率的条件下，处理成本相对低廉。

d、设备安装简单，开停机、运行方便，工艺流程简洁流畅，可实现PLC自动控制；

e、获得亲电加成生成的 · OH的共同作用并利用空气中的氧气参与链式反应，视具体水质情况可不投加 H2O2或降低H2O2和亚铁的消耗量；

第二、与传统Fenton法对比：

a、降低原来的酸碱调节，降低酸碱化料投加成本；

b、获得亲电加成生成的自由基利用空气中的氧气参与链式反应，降低了H2O2的消耗量；

c、引入多金属系催化剂及同相异相催化氧化技术，提高氧化剂利用效率；

d、产生的三价铁大部分得以结晶或沉淀披覆在反应器内，从而减少亚铁的用量；

e、反应后沉淀物（产泥量）较传统Fenton法减少；

应用领域

实用新型多级氧化反应技术是一项更实用性和经济性的改良Fenton法，它产生的羟基自由基（ · OH） 是自然界中氧化性第二强的基团（仅次于氯离子），基本上能够氧化所有的物质。恒邦洁公司所有的四相多级氧化反应器可以根据 业主单位的需求，帮助业主解决一般工艺出水CODcr任然不能达标的问题。

迄今为止该技术和相位设备在国内多家制药企业、化工企业、印染企业等的废水处理深度处理中得到应用，可以根据业主单位 水质水量的情况设计合适的体积尺寸，灵活应用在新建项目或者改造项目的相应的工艺单元位置。

概述

随着国家经济的发展，对于生产水资源的需求越来越大，而水资源的缺乏使得各个行业对中水回用的需求也随之增大，尤其是采油厂、 焦化和印染行业，因此，中水回用率越来越受到人们的关注。我公司开发出一种既经济又效益的处理方法，工艺流程为：“前吸附 + 一次 沉淀池 +　BAF1 + BAF2 + BAF3 + 后吸附 + 二沉淀池 + HBJ连续流砂滤器”。该技术已在采油厂、焦化厂和印染厂等多个厂区中水回用项目 中得到验证，目前已向各个行业的中水回用项目积极推广。

整体优势

1、采用改良式的曝气生物滤池：相比普通活性污泥法整体停留时间短，对水质适应性强，耐冲击性能好，调试启动速度快，同时具备生物氧化功能和过滤功能

2、通过前后吸附的方法，以保证生化段进水水质和出水指标。

3、HBJ连续流砂滤器系统采用升流式流动床过滤和单一均质石英砂滤料，过滤与洗砂同时进行，能够24小时连续自动运行，无需停机反冲洗， 巧妙的提砂和洗砂结构代替了传统大功率反冲洗系统，能耗极低，系统无需维护，管理简便，可无人值守

我公司在常规河道治理技术基础上勇于创新开发，根据城市内河河道形态特点以及水体受污染情况，结合自身技术优势，以载体固定酶 - 高效微生物技术为核心， 采用“排污口预处理 + 主体水质提升 + 底泥稳定”治理模块，以根治河道污染，保证水质改善。同时专有的HBJ-河道治理一体化设备可有效治理污染严重的区域及排污口， 提高水体溶氧，去除CODcr、色度、臭味。

载体固定酶 - 高效微生物技术

载体固定酶 - 高效微生物技术在河水处理过程能够把有机物彻底分解，然后产生CO2、H2O，因此整个处理过程不产生污泥、 臭味等二次污染。

1、排污口预处理

2、主体水质提升

主体水质提升以“载体固定酶 - 高效微生物技术”为核心，高效净化水质，同时结合特定水生植物，提高河道景观， 并协同水质净化，达到治理目标。

概述

HBJ-BAF是公司自主研发的曝气生物滤池，是在固定化微生物技术(IM)的基础上，结合曝气生物池发展而成的污水处理新工艺。与传统的曝气生物滤池相比，HBJ-BAF采用一种特殊的具有大孔性网状结构的载体，载体的表面具有特殊化学性质，能牢固地吸附固定水中的微生物。开孔采用大孔与微孔相接合的方式，大孔保持良好的气、液、固的接触条件，三相传质推动力大大增加，微孔用于固定化微生物。同时，通过分子设计，在载体引入大量的活性和强极性基团并通过固定化技术，将大量变异菌和酶制剂牢牢固定在载体上，单位体积生物量大、最高可达60g/L。

工艺特点

a、所有载体比表面积达80m2/g，孔隙率为98%，载体平均湿密度为1.00g/cm3，在水中呈悬浮状。

b、具有抗冲击、负荷高的特点，在一定浓度范围内污染物去除率随CODcr容积负荷的增大而升高。尤其对处理高难度的有机废水有独特的效果。

c、HBJ-BAF污水处理工艺中的污水氨氮含量在1000mg/L以下时，未发现对硝化菌与其它菌种有抑制现象，突破了午睡中的氨氮含量高于200mg/L时将对微生物产生抑制的结论，达到国际先进水平。

d、采用高效微生物，反应速度是传统曝气的数倍。温度适应范围广，尤其低温下具有较好的生物活性，能保证处理结果，可在高寒地区应用。

e、污染物去除率高、占地面积省、投资运行成本低、出水水质好，管理方便。

概述

HBJ-MBR是一种由膜分离单元与生物处理单元相结合的新型水处理技术。采用的膜结构主要为恒邦洁公司自主研发的HBJ平板膜和HBJ中空纤维膜，按膜孔径可划分为微滤、超滤、纳滤、反渗透技术。

主要特点

a、出水水质稳定，耐冲击负荷高
b、污泥龄长，剩余污泥产量少
c、占地面积小，不受场地限制
d、氨氮及难降解有机物的去除效率高
e、操作管理方便，易于实现自动控制
f、易于从传统工艺进行改造
